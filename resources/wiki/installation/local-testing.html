Download the latest papiro version from the official github repository. In case your testing on a local machine, just enter the repository directory using the following command.

cd ./papiro

Once there the typescript must be transpiled using the following command.

tsc

Use the following command to launch a new webserver.

python ./server-n-cache.py

Once done, the web will be launched in the following URL.

http://127.0.0.1:8000

*** Warning: You need to add your markdown or html files to the web. ***

As you are in a local environment, there is no pipeline to automatically convert your files and index them in real time. Copy your files (folders and html or markdown files) to the following path.

resources/uploads

Then use the papiro indexer binary to convert and index your files

./bin/papiro-indexer -path ./resources/uploads -destination ./resources/wiki
